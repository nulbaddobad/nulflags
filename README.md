## PT-BR

### Como usar minhas fastflags

1. Baixe "fastflag_injector_gui_enhanced.exe"  
2. Baixe as fastflags  

### Como injetar minhas fastflags

1. Abra "fastflag_injector_gui_enhanced.exe", clique em [Load JSON], selecione o arquivo da fastflag, depois clique em [Start Watching], [Stop] e [Inject Once]  
2. Certifique-se de que o Roblox está aberto na página inicial  
3. Entre em um jogo e aproveite

# README

## EN-US

### How to use my fastflags

1. Download "fastflag_injector_gui_enhanced.exe"  
2. Download the fastflags  

### How to inject my fastflags

1. Open "fastflag_injector_gui_enhanced.exe", click [Load JSON], select the fastflag file, then click [Start Watching], [Stop], and [Inject Once]  
2. Make sure Roblox is open and on the home page  
3. Open a game and enjoy
