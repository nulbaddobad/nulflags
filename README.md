# README

## How to use my fastflags

1. Download "fastflag_injector_gui_enhanced.exe"  
2. Download the fastflags  

## How to inject my fastflags

1. Open "fastflag_injector_gui_enhanced.exe", click [Load JSON], select the fastflag file, then click [Start Watching], [Stop], and [Inject Once]  
2. Make sure Roblox is open and on the home page  
3. Open a game and enjoy
